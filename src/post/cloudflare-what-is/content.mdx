## 企業としての Cloudflare（クラウドフレア）

Cloudflare, Inc.（クラウドフレア）はアメリカの企業。インターネットの速度とセキュリティを向上させるサービスを提供。

主要なサービスは以下の通り：

- **コンテンツデリバリーネットワーク（CDN）**：ウェブサイトのコンテンツを迅速に配信。
- **インターネットセキュリティ**：ウェブサイトの安全を確保。
- **DDoS 防御**：サイバー攻撃からの保護。
- **分散型ドメイン名サーバシステム**：インターネットアドレスの管理と迅速なアクセス。

## サービスとしての Cloudflare（クラウドフレア）

### DDoS 攻撃対策

- **Under Attack（アンダーアタック）モード**：
  - サイトにアクセスする訪問者に JavaScript のチャレンジ（計算課題）をして不正アクセスを防止。
- **Web アプリケーションファイアウォール**：
  - ウェブサイトを不正アクセスから守る。

### DNS

- **ドメインネームサーバー（DNS）**：ウェブサイトのアドレスを管理。
- **1.1.1.1**：高速かつプライバシーを重視した無料 DNS サービス。
- **1.1.1.1 for Families**：不適切なコンテンツやマルウェアをブロック。

### リバースプロキシ

- ウェブトラフィックを中継し、ウェブサイトの速度とセキュリティを向上。

### コンテンツ配信ネットワーク

- コンテンツを世界中にキャッシュし、迅速に配信。

### レジストラサービス

- **Cloudflare Registrar**
- ドメイン名の登録と管理。
- ドメインを原価で提供している。
  - ドル円の為替レートの影響は受けるが、それでも他よりも安く購入できるケースが多い。
  - 以前までは新規ドメイン取得をできなかったが、現在では新規で独自ドメイン取得が可能。

### メールセキュリティ

- カスタムメールアドレス作成、メールルーティング管理、メールセキュリティ保護。
