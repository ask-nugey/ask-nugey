## MacでWindows環境を無料で構築する方法

以下のように様々ある

- VirtualBox: 仮想環境
- UTM: 仮想環境
- Parallels: 仮想環境
- Boot Camp: Intel Macのみ
- Azureで構築して接続: クラウド
- さくらのVPS for Windows Server: クラウド
- お名前.com デスクトップクラウド: クラウド
- VMware Fusion: 仮想環境

## VMware Fusion で構築

VMware Fusion と VMware Workstation は2024/11にpro版、商用利用含め無料になった  
サッとWindowsで動作確認したい場合はこれで十分

https://blogs.vmware.com/cloud-foundation/2024/11/11/vmware-fusion-and-workstation-are-now-free-for-all-users/

- VMware Fusion:
  - ホストOSが Mac ならこっちを使用
- VMware Workstation: 
  - ホストOSが WindowsまたはLinux ならこっちを使用

### Windows 仮想マシンの作成

今回は、プロダクトキーやインストールメディアを持っていない状況を想定し、新規に仮想マシンを作成します。

まず、「Microsoft から Windows を取得」を選択します。

![](https://qiita-user-contents.imgix.net/https%3A%2F%2Fqiita-image-store.s3.ap-northeast-1.amazonaws.com%2F0%2F864151%2F88024511-087b-17aa-7253-0aacd93f1dc7.png?ixlib=rb-4.0.0&auto=format&gif-q=60&q=75&s=3889eedcbac276974b96c3510495a33e)

![](https://qiita-user-contents.imgix.net/https%3A%2F%2Fqiita-image-store.s3.ap-northeast-1.amazonaws.com%2F0%2F864151%2F563ff4ae-9817-9fe7-110b-0e151dd1855d.png?ixlib=rb-4.0.0&auto=format&gif-q=60&q=75&s=0a9c88e1076bfb447f249cd384c47591)

事前に Windows 11 のイメージを準備しておく必要はなく、VMware Fusion から直接ダウンロードできます。

![](https://qiita-user-contents.imgix.net/https%3A%2F%2Fqiita-image-store.s3.ap-northeast-1.amazonaws.com%2F0%2F864151%2F2f0f182a-4276-597d-045c-9475554dfad1.png?ixlib=rb-4.0.0&auto=format&gif-q=60&q=75&s=d0da94a599baaf7c0b6932b062e44dfe)
