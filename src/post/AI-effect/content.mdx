## AI効果とは
AI効果（AI Effect）とは、人工知能が新しい課題を達成するたびに、人々がその課題を「真の知能」の定義から除外してしまう傾向を指す現象のことです。  
簡単にいうと「AIができるようになったことは、もはや知能とは見なされない」という人間の心理現象です。

## AI効果が生まれる背景
この現象の背景には、「知能」に対する人間の理解の変化があります。  
人は常に「真の知能」を現在のAIにはまだできていないことと定義しようとする傾向があります。これは"人間の知能の特殊性や優位性を保持したい"という無意識的な心理が働いているためと考えられています。

## AI効果の具体例
Aチェスは最も顕著な例の一つです。  
1997年にIBMのディープブルーがチェス世界チャンピオンのガルリ・カスパロフに勝利しました。

| ゲーム # | 白                   | 黒               | 結果 | 備考                     |
| -------- | -------------------- | ---------------- | ---- | ------------------------ |
| 1        | **カスパロフ**       | ディープ・ブルー | 1–0  |                          |
| 2        | **ディープ・ブルー** | カスパロフ       | 1–0  |                          |
| 3        | カスパロフ           | ディープ・ブルー | ½–½  | 双方の同意の上の引き分け |
| 4        | ディープ・ブルー     | カスパロフ       | ½–½  | 双方の同意の上の引き分け |
| 5        | カスパロフ           | ディープ・ブルー | ½–½  | 双方の同意の上の引き分け |
| 6        | **ディープ・ブルー** | カスパロフ       | 1–0  |                          |

　  
その際、多くの人は「チェスは単なる計算の組み合わせであり、真の知能を必要としない」と主張し始めました。同様の現象は、画像認識、自然言語処理、音声認識など、AIが進歩するたびに繰り返し観察されています。

## AI効果からわかること
AI効果は、技術の進歩に対する人間の認識や評価の変化を示す重要な社会現象です。  
この効果は、以下のようなことを提唱していると言えるでしょう↓

1. 人間の知能の定義は固定的ではなく、技術の進歩とともに常に変化している
2. AIの進歩は、人間の特殊性や知能の本質についての再考を促している
3. 技術の評価基準は、その技術が一般化するにつれて変化していく
